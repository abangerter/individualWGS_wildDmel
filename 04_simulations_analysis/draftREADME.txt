### DRAFT README 


### SUMMARIZING THE SIMULATIONS 
	## step 1: aggregate the information 
		The script gatherSims_and_calculate_FIS.R takes the raw output from the simulations and calculates FIS on each SNP, each iteration, and each "population" for both the simulated observed and simulated true genotypes. 

		The script get_empirical_FIS.R takes the empirical dataset and calculates per-SNP FIS estimates. It requires input files from scripts getSeqArrayGDSfile.R and getHeterozygosityInfo.R

	## step 2: get summary estimates 
		The script get_average_FIS_estimates.R takes the files generated with the script gatherSims_and_calculate_FIS.R to generate average FIS estimates per simulation iteration for the simulations and the empirical data. This script will require output files generated by scripts gatherSims_and_calculate_FIS.R, get_empirical_FIS.R, and getLDprunedSNPset.R


### ANALYSIS
	## JACKKNIFE
		The script jackknife_analysis.R performs FIS estimates in a jackknife fashion, dropping each individual per time point once. This script will require output files generated by getHeterozygosityInfo.R, getSeqArrayGDSfile.R, and getLDprunedSNPset.R

	## REMOVAL OF INVERSION HOMOZYGOTES
		The script FIS_without_inv_homozygotes.R recalculates per-SNP FIS after removal of inversion homozygotes, then calculates the inversion-homozygote-free average FIS. This script will require raw output files generated in the simulations, output files made by scripts getHeterozygosityInfo.R, getSeqArrayGDSfile.R, and getLDprunedSNPset.R, and the list of inversion homozygotes identified by script inversionTyping_fig02.R

	## SLIDING WINDOW 
		The script build_startStop_array.R generates the position array used for starts and stops in the sliding window analysis. It requires files generated by get_empirical_FIS.R and getLDprunedSNPset.R

		The script build_argument_array.R generates the array of arguments used in the sliding window analysis. 

		The script slidingWindowFis.R is the script that runs the sliding window analysis on the simulations, and was submitted as an array on a computing cluster with submitSWscript.slurm using the array of arguments generated by build_argument_array.R. This script will require ouput files generated by gatherSims_and_calculate_FIS.R and build_startStop_array.R

		The script slidingWindowFis_data.R performs the sliding window analysis on the empirical data. This script will require ouput files generated by build_startStop_array.R, getLDprunedSNPset.R, and get_empirical_FIS.R



### FIGURES

	The script figure_07.R generates Figure 07. It requires files generated by script FIS_without_inv_homozygotes.R and get_average_FIS_estimates.R

	The script figure_09.R generates Figure 09. It requires files generated by slidingWindowFis.R and slidingWindowFis_data.R
